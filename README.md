Bu proje, Web Tasarım ve Programlama dersi dönemi final ödevi kapsamında, Petrol Ofisi'nin kurumsal kimliğine ve modern web standartlarına uygun olarak sıfırdan geliştirilmiş bir çoklu sayfa (Multi-page) web portalı simülasyonudur.

## Kullanılan Teknolojilerin Özeti

Proje, dışarıdan hiçbir hazır şablon, CSS kütüphanesi (Bootstrap, Tailwind vb.) veya JavaScript framework'ü (React, Vue vb.) kullanılmadan, tamamen Saf (Vanilla) Web Teknolojileri ile performans odaklı geliştirilmiştir:

* HTML: Sitenin semantik (anlamlı) iskeleti kurgulanmış; SEO ve erişilebilirlik standartlarına uygun etiket mimarisi (nav, section, article, footer) tercih edilmiştir.
* CSS: Flexbox ve CSS Grid modelleri kullanılarak kurumsal renk paletiyle tasarlanmış, tamamen mobil uyumlu (Responsive) esnek arayüz.
* JavaScript: Kullanıcı deneyimini dinamikleştiren tüm bileşenler saf JavaScript kodları ile asenkron biçimde programlanmıştır.

---

## Projedeki JavaScript Etkileşimleri

* Dinamik Kahraman Slaytı (Hero Slider): Görsel geçişlerini ve aktif kontrol noktalarını (dots) dinamik yöneten döngü mekanizması. Kullanıcı deneyimini (UX) artırmak adına, imleç slider üzerine geldiğinde otomatik geçişi durduran (mouseenter) ve ayrıldığında tekrar başlatan (mouseleave) akıllı olay dinleyicileri entegre edilmiştir.
* Sürükle-Kaydır Kampanya Paneli (Drag-to-Scroll): Kampanyalar alanında, kullanıcıların fareyle basılı tutarak paneli pürüzsüzce sağa sola kaydırabilmesini sağlayan dinamik mesafe hesaplamalı (walk) ve fare hareket takipli (mousedown, mousemove, mouseup) sürükleme simülasyonu yazılmıştır.
